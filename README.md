# Cpp02

42 C++ module 02 exercises focused on fixed-point numbers and operator overloading.

## Structure

- `ex00`: basic fixed-point class with constructors, copy behavior, and raw bit access
- `ex01`: adds float and integer conversions plus stream output
- `ex02`: adds arithmetic, comparison, increment/decrement, and min/max helpers

## Build

Each exercise is self-contained.

```sh
cd ex00 && make
cd ex01 && make
cd ex02 && make
```

## Notes

The repository now ignores common build artifacts so only the source files are committed.
